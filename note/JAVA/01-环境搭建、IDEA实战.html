Java 从零开始：环境搭建、IDEA 实战与语法详解 ​

作为编程语言中的"常青树"，Java 凭借其跨平台性、安全性和强大的生态系统，在企业级开发、移动应用（Android）、大数据等领域占据重要地位。本文将从零基础开始，带你完成 Java 环境搭建、掌握 IntelliJ IDEA 的核心用法，并系统学习 Java 基础语法，附带大量实操截图和代码示例，助你快速入门。

一、Java 开发环境搭建（Windows + Mac 双平台） ​

1. 什么是 JDK？为什么需要它？ ​

Java 程序的运行依赖于 JDK（Java Development Kit，Java 开发工具包），它包含：

  • JRE（Java Runtime Environment）：Java 运行环境，负责执行.class 文件
  • 编译器（javac）：将.java 源代码编译为.class 字节码
  • 调试工具、文档工具等开发必需组件

简单说：没有 JDK，无法开发 Java 程序；没有 JRE，无法运行 Java 程序。

2. JDK 下载与安装（以 JDK 17 为例，LTS 长期支持版） ​

（1）下载地址 ​

官网：Oracle JDK（需注册账号）
备选：Adoptium OpenJDK（开源免费，推荐新手）

（2）Windows 安装步骤 ​

  1. 选择对应版本（x64 Installer），下载后双击运行
  2. 安装界面保持默认路径（建议：C:\Program Files\Java\jdk-17），点击"下一步"直至完成

（3）Mac 安装步骤 ​

  1. 选择 macOS 版本（.dmg 格式），下载后打开，双击.pkg 文件
  2. 跟随向导完成安装，默认路径：/Library/Java/JavaVirtualMachines/jdk-17.jdk/Contents/Home

3. 环境变量配置（关键步骤） ​

（1）Windows 配置 ​

  1. 右键"此电脑"→"属性"→"高级系统设置"→"环境变量"
  2. 在"系统变量"中点击"新建"，设置：
    • 变量名：JAVA_HOME
    • 变量值：JDK 安装路径（如 C:\Program Files\Java\jdk-17）
  3. 找到Path变量，点击"编辑"，新增两条路径：
    • %JAVA_HOME%\bin
    • %JAVA_HOME%\jre\bin（部分 JDK 版本可能无 jre 目录，可忽略）

（2）Mac 配置 ​

  1. 打开终端，输入命令编辑配置文件（根据 shell 类型选择）：

4. 验证环境是否配置成功 ​

打开终端（Windows 用 PowerShell/CMD，Mac 用终端），输入以下命令：

bash

javac -version  # 查看编译器版本

二、IntelliJ IDEA 详细使用指南（社区版免费够用） ​

1. 为什么选择 IntelliJ IDEA？ ​

IDEA 是 JetBrains 公司推出的 Java 集成开发工具，相比 Eclipse 具有：

  • 更智能的代码提示和补全
  • 内置 Git、Maven 等工具，无需额外配置
  • 强大的重构功能和调试工具
  • 对 Kotlin、Scala 等 JVM 语言友好支持

社区版（Community）免费，足够学习和开发普通 Java 项目；旗舰版（Ultimate）收费，增加了 Spring、数据库等企业级功能。

2. IDEA 下载与安装 ​

  1. 官网下载：IntelliJ IDEA，选择"Community Edition"
  2. 安装步骤：
    • Windows：双击.exe，勾选"Add launchers dir to PATH"（方便终端启动），其余默认
    • Mac：拖拽到 Applications 文件夹即可

3. 首次启动与配置 ​

  1. 首次启动会提示导入配置，选择"Do not import settings"
  2. 选择主题（推荐 Darcula 深色主题，保护视力）
  3. 插件安装（可选）：
    • 点击"Plugins"，搜索并安装"Chinese (Simplified)"（中文插件）
    • 其他推荐：Lombok（简化代码）、CheckStyle（代码规范检查）

4. 创建第一个 Java 项目 ​

  1. 点击"New Project"，选择"Java"，右侧选择已安装的 JDK（若未检测到，点击"Add JDK"手动指定路径）
  2. 填写项目名称（如FirstJavaProject）和路径，点击"Create"
  3. 项目结构说明：
    • src：存放源代码（.java 文件）
    • External Libraries：项目依赖（此处为 JDK 类库）

5. 核心界面与常用功能 ​

（1）创建 Java 类 ​

右键src→"New"→"Java Class"，输入类名（如HelloWorld，首字母大写），生成代码框架：

java
public class HelloWorld {
}

（2）编写并运行第一个程序 ​

在类中添加main方法（程序入口），代码如下：

java
public class HelloWorld {
    public static void main(String[] args) {
        System.out.println("Hello, Java!");  // 打印语句
    }
}

  • 点击代码左侧的绿色三角图标
  • 右键代码区→"Run 'HelloWorld.main()'"
  • 快捷键：Ctrl+Shift+F10（Windows）/Control+Shift+R（Mac）

（4）调试功能（排错必备） ​

  1. 在代码行号左侧点击设置断点（红色圆点）
  2. 点击"Debug"按钮（绿色虫子图标），程序会在断点处暂停
  3. 调试窗口功能：
    • 下一步（Step Over）：F8，执行当前行并跳到下一行
    • 进入方法（Step Into）：F7，进入当前调用的方法内部
    • 继续执行（Resume Program）：F9，运行到下一个断点
