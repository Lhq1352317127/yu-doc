01-CSS基础练习：JD首页的制作（顶部和底部） ​

前言 ​

京东作为国内领先的电商平台，其官网设计极具代表性，是前端学习的经典案例。本文将带你使用 WebStorm 开发工具，从零开始搭建京东首页的顶部与底部结构，掌握网页布局、CSS 初始化、浮动与清除、精灵图使用等核心技能。

京东首页参考图：http://img.smyhvae.com/20180119_1653.jpg

页面规划：创建项目工程 ​

首先，在 WebStorm 中新建一个空白项目，结构如下：

jd-project/
├── css/
│   └── base.css
├── images/
│   ├── logo.png
│   ├── topbanner.jpg
│   ├── sprite.png
│   └── ...
└── index.html

CSS 初始化（Reset 与公共样式） ​

为确保页面在不同浏览器中表现一致，需进行 CSS 初始化。将以下代码保存至 css/base.css：

css
@charset "UTF-8";
/* CSS 初始化 */
html, body, ul, li, ol, dl, dd, dt, p, h1, h2, h3, h4, h5, h6, form, fieldset, legend, img {
    margin: 0;
    padding: 0;
}
fieldset, img, input, button {
    border: none;
    padding: 0;
    margin: 0;
    outline: none;
}
ul, ol {
    list-style: none;
}
input {
    padding-top: 0;
    padding-bottom: 0;
    font-family: "SimSun", "宋体";
}
select, input {
    vertical-align: middle;
}
select, input, textarea {
    font-size: 12px;
    margin: 0;
}
textarea {
    resize: none;
}
img {
    border: 0;
    vertical-align: middle;
}
table {
    border-collapse: collapse;
}
body {
    font: 12px/150% Arial, Verdana, "\5b8b\4f53";
    color: #666;
    background: #fff;
}

/* 清除浮动（推荐方式） */
.clearfix:before,
.clearfix:after {
    content: "";
    display: table;
}
.clearfix:after {
    clear: both;
}
.clearfix {
    *zoom: 1; /* IE6/7 */
}

/* 全局链接样式 */
a {
    color: #666;
    text-decoration: none;
}
a:hover {
    color: #C81623;
}
h1, h2, h3, h4, h5, h6 {
    font-weight: normal;
    font-size: 100%;
}
s, i, em {
    font-style: normal;
    text-decoration: none;
}
.col-red {
    color: #C81623 !important;
}

/* 公共类 */
.w {
    width: 1210px;
    margin: 0 auto;
}
.fl { float: left; }
.fr { float: right; }
.ac { text-align: center; }
.hide { display: none; }

说明：

  • outline: none 移除输入框默认蓝边。
  • vertical-align: middle 解决图片底部 3px 空隙。
  • .clearfix 使用伪元素清除浮动，兼容性好。
  • .w 为页面版心，固定宽度居中。

引入 CSS 与 Favicon ​

在 index.html 中引入样式与网站图标：

html
<!DOCTYPE html>
<html lang="zh-CN">
<head>
    <meta charset="UTF-8">
    <title>京东首页 - 顶部与底部</title>
    <!-- 引入 CSS -->
    <link rel="stylesheet" href="css/base.css">
    <!-- 引入 Favicon -->
    <link rel="shortcut icon" href="favicon.ico">
</head>
<body>
    <!-- 页面内容 -->
</body>
</html>
