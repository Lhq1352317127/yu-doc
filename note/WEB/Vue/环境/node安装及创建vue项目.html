一、node安装 ​

1、解压node文件夹 ​

解压好的文件就是这样了

2、配置环境 ​
1 )、配置node环境变量 ​

然后打开【系统环境变量】中的【Path】

把刚刚解压的node-v16.17.1-win-x64的路径配置到系统环境变量中

2）、然后查看是否配置成功 ​

打开cmd（命令窗口）

使用这个目录查看是否配置成功
node -v
1
2

如果出现了版本号就是成功了

3）、创建文件夹及配置环境变量 ​

创建两个文件夹：node_global、node_cache

然后配置node_global文件夹的路径到系统环境变量中

然后打开系统环境变量配置到【系统环境变量】中的【Path】

创建完毕后，使用管理员身份打开cmd命令窗口（打开方法见下方Tips），输入

npm config set prefix “你的路径\node_global” （复制你刚刚创建的“node_global”文件夹路径）

npm config set prefix "D:\develop\Node.js\node_global"
1

npm config set cache “你的路径\node_cache” （复制你刚刚创建的“node_cache”文件夹路径）

  npm config set cache "D:\develop\Node.js\node_cache"
1

然后再node_global目录中创建 node_modules 这个目录

然后把node_modules 的路径配置到【系统环境变量】中的【Path】

4）、安装vue脚手架（vue@cli） ​

在cmd命令窗口中安装vue脚手架输入npm install -g vue@cli，等待下载

npm install -g vue@cli
1
