Vue 全家桶核心知识点整理 ​

1. Vue 基础 ​

1.1 Vue 的特点 ​

  • 遵循 MVVM 模式
  • 编码简洁，体积小，运行效率高，适合移动/PC 端开发
  • 专注于 UI 层，可结合第三方库扩展功能
  • 借鉴 Angular 的模板和数据绑定技术

1.2 核心语法 ​

1.2.1 插值语法与指令语法 ​

  • 插值语法：，用于解析标签体内容，xxx 作为 JS 表达式解析
  • 指令语法：以 v- 开头，用于解析标签属性、标签体内容或绑定事件，例如 v-bind:href="url"（可简写为 :href="url"）

1.2.2 数据绑定 ​

  • 单向数据绑定：v-bind（简写 :），数据从 data 流向页面
  • 双向数据绑定：v-model，数据可在 data 与页面间双向流动（仅用于表单元素）
    html
    <input v-model="username" />
<!-- 等价于 :value="username" @input="username=$event.target.value" -->
    1
    2

1.3 事件处理 ​

  • 绑定方式：
    • v-on:事件名="函数" 或简写 @事件名="函数"
    • 传参：@事件名="函数(参数)"
  • 事件修饰符：
    • .prevent：阻止默认行为（如表单提交）
    • .stop：阻止事件冒泡
    • 可链式使用：@click.prevent.stop="handleClick"
  • 按键修饰符：
    • 按键码：@keyup.13="submit"（13 对应 Enter 键）
    • 按键名：@keyup.enter="submit"

1.4 计算属性与监视属性 ​

  • 计算属性（computed）：

    • 用于复杂数据计算，基于依赖自动更新
    • 缓存计算结果，性能优于方法
    javascript
    computed: {
  fullName() {
    return this.firstName + '-' + this.lastName
  }
}

1.5 样式绑定 ​

  • class 绑定：

    • 字符串：:class="className"
    • 对象：:class="{ active: isActive, 'text-danger': hasError }"
    • 数组：:class="['active', 'text-danger']"

  • style 绑定：

    html
    <div :style="{ color: activeColor, fontSize: fontSize + 'px' }"></div>
    1

1.6 条件渲染与列表渲染 ​

  • 条件渲染：

    • v-if/v-else：通过创建/删除 DOM 元素控制显示（适合不频繁切换场景）
    • v-show：通过控制 display 样式控制显示（适合频繁切换场景）

  • 列表渲染：

    • v-for：遍历数组/对象，语法 v-for="(item, index) in list"
    • 必须绑定 key：提高渲染效率，建议使用唯一标识（如 ID）
    html
    <ul>
  <li v-for="(person, index) in persons" :key="person.id">{{ index }}. {{ person.name }}-{{ person.age }}</li>
</ul>

1.7 表单数据收集 ​

  • 文本框/textarea：v-model 绑定字符串
  • 单选框：v-model 绑定选中值（value）
  • 复选框：单个绑定布尔值，多个绑定数组
  • 下拉框：v-model 绑定选中项的 value

1.10 生命周期 ​

  • 主要阶段：
    1. 创建阶段：beforeCreate → created（可发起 Ajax 请求）
    2. 挂载阶段：beforeMount → mounted（操作 DOM）
    3. 更新阶段：beforeUpdate → updated
    4. 销毁阶段：beforeDestroy（收尾工作，如清除定时器）→ destroyed
  • 常用钩子：created（初始化数据）、mounted（操作 DOM）、beforeDestroy（清理资源）

2. 组件化编程 ​

2.1 核心概念 ​

  • 模块：提供特定功能的 JS 程序（通常为单个 JS 文件）
  • 组件：实现局部功能的代码集合（HTML/CSS/JS 等）
  • 模块化：应用由多个模块组成
  • 组件化：应用由多个组件组成

4. Vue Router ​

4.1 基本概念 ​

  • 用于 SPA（单页应用）的路由管理，通过 hash 或 history 模式实现无刷新跳转

4.2 基本使用 ​

  1. 定义路由组件

  2. 配置路由规则：

    javascript
    import Vue from 'vue';
import VueRouter from 'vue-router';
import Home from './Home.vue';
import About from './About.vue';

Vue.use(VueRouter);

const routes = [
  { path: '/', redirect: '/home' }, // 重定向
  { path: '/home', component: Home },
  { path: '/about', component: About }
];

const router = new VueRouter({ routes });
export default router;

5. Vuex ​

5.1 概念 ​

  • 集中式状态管理库，用于多组件共享数据

5.2 核心概念 ​

  • state：存储状态（类似 data）

    javascript
    const state = { count: 0 };
    1

  • mutations：修改 state 的唯一方式（同步操作）

    javascript
    const mutations = {
  increment(state, payload) {
    state.count += payload;
  }
};
    1
    2
    3
    4
    5

  • actions：处理异步操作，通过 commit 调用 mutations

    javascript
    const actions = {
  incrementAsync({ commit }, payload) {
    setTimeout(() => {
      commit('increment', payload);
    }, 1000);
  }
};
    1
    2
    3
    4
    5
    6
    7

  • getters：计算派生状态（类似 computed）

    javascript
    const getters = {
  doubleCount(state) {
    return state.count * 2;
  }
};
    1
    2
    3
    4
    5

  • modules：拆分模块，解决状态臃肿

    javascript
    const store = new Vuex.Store({
  modules: {
    user: userModule,
    cart: cartModule
  }
});

// 调用 mutations（不推荐直接调用，建议通过 actions）
