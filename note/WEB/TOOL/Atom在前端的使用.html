方法一：设置代理 ​

C:\Users\smyhvae\.atom\.apm目录下的.apmrc配置文件（没有就新建一个）,然后添加代理信息：

strict-ssl=false
https-proxy=http://127.0.0.1:1080/
http-proxy =http://127.0.0.1:1080/
1
2
3

这里的 http://127.0.0.1:1080，是我自己的 Shadowsocks 代理，你需要设置成自己的可用代理。 然后再执行：

apm install --check
1

应该可以测试成功，祝好运~~
