npm、yarn、pnpm 设置最新国内镜像源（2024更新） ​

一、背景说明 ​

阿里于2024年1月22日更新淘宝镜像域名，原registry.npm.taobao.org证书过期，新镜像为registry.npmmirror.com。

二、包管理器镜像设置 ​

1. npm ​

bash
# 查询当前源
npm get registry

# 设置国内镜像（推荐）
npm config set registry https://registry.npmmirror.com/

# 恢复官方源
npm config set registry https://registry.npmjs.org/

四、使用nrm管理镜像（推荐） ​

1. 安装 ​

bash
npm install nrm -g
1

2. 常用命令 ​

bash
# 查看所有镜像（* 表示当前使用）
nrm ls

# 切换镜像（示例：淘宝源）
nrm use taobao

# 测试镜像速度
nrm test

# 添加自定义镜像
nrm add [name] [url]

# 删除镜像
nrm del [name]

五、常见问题 ​

1. 清除缓存 ​

bash
npm cache clean --force
1

2. yarn不生效问题 ​

  • 原因：yarn v1.x优先读取.yarnrc，需手动修改
  • 解决方案：
    bash
    # 直接修改配置
yarn config set registry https://registry.npmmirror.com/
    1
    2

3. nrm原理 ​

nrm修改的是用户目录下的.npmrc文件，通过覆盖registry字段实现镜像切换。
